# Plan: Subprocess LLM Drivers for Claude Code + Codex CLI

## Context

OpenFang agents need LLM access, but the user authenticates via **Claude Code OAuth** and **Codex CLI ChatGPT OAuth** — neither token works against standard API endpoints (`api.anthropic.com` rejects OAuth, `api.openai.com` returns "insufficient_quota" for ChatGPT tokens). The solution: spawn the CLI binaries as subprocesses, since both `claude -p` and `codex exec` handle auth internally.

**Verified working:**
- `claude -p "prompt" --output-format json` → returns JSON with `result`, `usage`, `total_cost_usd`
- `codex exec "prompt" --json` → returns JSONL with `item.completed` events and `turn.completed` usage

## Changes

### 1. New file: `crates/openfang-runtime/src/drivers/claude_code.rs`

**ClaudeCodeDriver** struct (no fields needed — stateless subprocess spawner):

```rust
pub struct ClaudeCodeDriver;
```

**`complete()` implementation:**
1. Convert `CompletionRequest` messages → single prompt string (role-prefixed lines)
2. If `request.system` is set, pass `--system-prompt`
3. Spawn: `claude -p --output-format json --model <request.model>`
4. Pipe prompt to stdin, read stdout
5. Parse JSON: extract `result` → `ContentBlock::Text`, `usage.input_tokens` + `usage.output_tokens` → `TokenUsage`
6. Timeout: 120 seconds
7. On error: map to `LlmError::Api` or `LlmError::Http`

**No tool use** — CLI doesn't support OpenFang's tool protocol. Return text-only responses.
**No custom stream impl** — use default (wraps `complete()`).

### 2. New file: `crates/openfang-runtime/src/drivers/codex_cli.rs`

**CodexCliDriver** struct (no fields):

```rust
pub struct CodexCliDriver;
```

**`complete()` implementation:**
1. Convert messages → single prompt string
2. Spawn: `codex exec --json --model <request.model> "<prompt>"`
3. Read stdout JSONL, collect all `item.completed` events where `item.type == "agent_message"`
4. Concatenate text from agent_message items
5. Parse `turn.completed` event for `usage.input_tokens` + `usage.output_tokens`
6. Timeout: 120 seconds
7. Codex has no `--system-prompt` flag — prepend system prompt to the user prompt if present

### 3. Edit: `crates/openfang-runtime/src/drivers/mod.rs`

- Add `pub mod claude_code;` and `pub mod codex_cli;`
- Add match cases in `create_driver()`:

  - `"codex"` → `Ok(Arc::new(codex_cli::CodexCliDriver))`
